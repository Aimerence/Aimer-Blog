# Aimerence Blog

🌟 基于 Astro 构建的个人知识库与博客，记录技术探索、生活感悟与认知成长

## ✨ 核心特色

## 核心框架

- Astro (静态生成框架)
- React (交互组件)
- TypeScript (类型安全)

## 样式系统

- Tailwind CSS (原子化样式)
- SCSS (预处理增强)
- PostCSS (样式优化)

## 内容生态

- Giscus (GitHub 评论系统)
- Fuse.js (模糊搜索)
- Mermaid (图表渲染)

## 工作流

- ESLint (代码质量)
- Prettier (代码格式化)
- GitHub Actions (CI/CD)

## 部署

- Vercel (边缘网络部署)
- Umami (隐私友好分析)
