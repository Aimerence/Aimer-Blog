这个博客使用了以下主要技术和框架：

## 核心框架

- Astro v4.16.18 - 静态站点生成器
- React v18.2.0 - 用于交互式组件
- TypeScript v5.5.3 - 类型安全的 JavaScript 超集

## 样式和UI

- Tailwind CSS v3.4.11 - 原子化 CSS 框架
- @tailwindcss/typography v0.5.15 - 优化文章排版
- 响应式设计 - 适配各种设备屏幕

## 功能模块

- @astrojs/rss v4.0.7 - RSS 订阅支持
- @giscus/react v3.0.0 - 基于 GitHub Discussions 的评论系统
- Fuse.js v7.0.0 - 全文搜索功能
- satori v0.11.0 - 动态生成 OG 图片
- @resvg/resvg-js v2.6.2 - SVG 渲染

## Markdown 增强

- remark-toc v9.0.0 - 自动生成目录
- remark-collapse v0.1.2 - 可折叠内容块

## 开发工具

- ESLint v8.56.0 - 代码质量检查
- Prettier v3.3.3 - 代码格式化
- @astrojs/check v0.9.3 - Astro 类型检查

## 主要功能

### 内容创作

- Markdown 写作支持
- 文章元数据管理
- 代码高亮
- 自动目录生成

### 用户体验

- 深色/浅色主题切换
- 响应式设计
- 全站搜索
- 文章阅读时间估算
- RSS 订阅

### 社交互动

- Giscus 评论系统
- 社交媒体分享
- 文章标签分类

### SEO 优化

- 自动生成 sitemap
- 动态 OG 图片
- meta 标签优化
